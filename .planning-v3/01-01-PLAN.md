# Plan 01-01: Lesson Generation + Board Shape Contract

## Overview
Build the AI lesson generation engine and the contract between lesson plans and tldraw board rendering. This is the foundation — everything else plugs into this.

## Context
- tldraw v4 is installed and verified (spike at src/components/whiteboard-spike/)
- V2 types exist at src/lib/classroom-v2/ — reference but don't import from them
- Claude Sonnet API key is available via ANTHROPIC_API_KEY env var
- tldraw v4 uses `toRichText()` for text in shapes, not plain string `text` prop
- Valid geo shape props: geo, dash, w, h, fill (none|semi|solid|pattern|fill|lined-fill), color, size, richText, font, align, verticalAlign, labelColor, url, growY
- Shapes are created via `editor.createShape({ id: createShapeId("teacher-xxx"), type: "geo"|"arrow"|"text", ... , isLocked: true })`

## Implementation

### Step 1: V3 Types (src/lib/classroom-v3/types.ts)
Define clean types for V3. Do NOT import from classroom-v2.

```typescript
// Core types needed:
// - LessonPlan: { lessonId, topic, gradeLevel, problems: LessonProblem[] }
// - LessonProblem: { problemId, problemText, boardShapes: BoardShape[], spokenPrompt, hints: string[], expectedAnswerType: "drawing"|"number"|"expression" }
// - BoardShape: tldraw-compatible shape definition { shapeId, type, x, y, props, isLocked }
// - ClassroomPhase: "idle"|"teaching"|"student_turn"|"evaluating"|"reviewing"|"ended"
// - SessionEvent: union type for all events that happen during a session
```

### Step 2: Lesson Generator (src/lib/classroom-v3/lesson-generator.ts)
Call Claude Sonnet API to generate a structured lesson plan.

- Input: { topic: string, gradeLevel: number } (e.g., "subtraction with regrouping", 4)
- Output: LessonPlan with 3 problems
- The prompt must instruct Claude to output board shapes as tldraw-compatible definitions
- Each problem gets: the math problem as text, positioned shapes for the board, a spoken prompt, hints for if the child gets stuck
- Use structured output (JSON mode) to get clean data back
- Use the Anthropic SDK (`@anthropic-ai/sdk`) — it should already be installed. If not, install it.

### Step 3: Board Renderer (src/lib/classroom-v3/board-renderer.ts)
Utility that takes a LessonProblem's boardShapes and renders them to a tldraw editor instance.

- `renderProblemToBoard(editor: Editor, problem: LessonProblem)`: clears teacher layer, draws all shapes
- `clearTeacherLayer(editor: Editor)`: removes all shapes with teacher- prefix
- `clearStudentLayer(editor: Editor)`: removes all shapes WITHOUT teacher- prefix
- All teacher shapes get `isLocked: true`
- Uses `editor.run(() => { ... }, { history: "ignore" })` for batched creation

### Step 4: API Route (src/app/api/lesson/generate/route.ts)
Simple POST endpoint that calls generateLessonPlan and returns the result.

### Step 5: Index barrel (src/lib/classroom-v3/index.ts)
Re-export everything from types, lesson-generator, board-renderer.
