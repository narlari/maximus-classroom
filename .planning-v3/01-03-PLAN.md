# Plan 01-03: TTS Voice

## Overview
Wire OpenAI TTS so the tutor speaks lesson narration aloud. When a lesson starts, the teacher "reads" the problem introduction. When the student moves to the next problem, the tutor narrates the transition.

## Context
- OpenAI client is already in use in the project (check existing routes for how it's imported)
- tldraw whiteboard is at /classroom-v3 (from 01-02)
- OPENAI_API_KEY is in .env.local — don't hardcode it
- ClassroomV3.tsx already has handleStartLesson and handleNextProblem — add speakText calls there
- Keep TTS non-blocking: don't await it in a way that freezes the UI

## Implementation

### Step 1: API route (src/app/api/tts/speak/route.ts)
- POST handler: reads `{ text }` from request body
- Calls OpenAI TTS: `openai.audio.speech.create({ model: "tts-1", voice: "nova", input: text })`
- Streams the response back as audio/mpeg
- Return a streaming Response with Content-Type: audio/mpeg

```ts
import OpenAI from 'openai'
import { NextRequest } from 'next/server'

const openai = new OpenAI()

export async function POST(req: NextRequest) {
  const { text } = await req.json()
  const speech = await openai.audio.speech.create({
    model: 'tts-1',
    voice: 'nova',
    input: text,
  })
  const buffer = Buffer.from(await speech.arrayBuffer())
  return new Response(buffer, {
    headers: { 'Content-Type': 'audio/mpeg' },
  })
}
```

### Step 2: Client helper (src/lib/classroom-v3/tts.ts)
```ts
export async function speakText(text: string): Promise<void> {
  try {
    const res = await fetch('/api/tts/speak', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ text }),
    })
    if (!res.ok) return
    const blob = await res.blob()
    const url = URL.createObjectURL(blob)
    const audio = new Audio(url)
    audio.play()
    audio.onended = () => URL.revokeObjectURL(url)
  } catch (e) {
    console.error('TTS error:', e)
    // Non-fatal — lesson continues without audio
  }
}
```

### Step 3: Wire into ClassroomV3.tsx
- Import `speakText` from `@/lib/classroom-v3/tts`
- In `handleStartLesson()`: after calling renderProblemToBoard, call:
  ```ts
  speakText(`Let's work on ${lesson.problems[0].problemText}`)
  ```
  Don't await — fire and forget so UI doesn't freeze
- In `handleNextProblem()`: after calling renderProblemToBoard, call:
  ```ts
  speakText(`Great! Now let's try: ${lesson.problems[nextIndex].problemText}`)
  ```
- When lesson ends (no more problems): `speakText("Amazing work! Lesson complete!")`

### Step 4: Export from index
Add to src/lib/classroom-v3/index.ts:
```ts
export { speakText } from './tts'
```
